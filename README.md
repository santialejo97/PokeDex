# Ejecutar en Desarrollo

1. Clonar el respositorio del proyecto que se encuentra en gitHub.
2. Ejecutar el comando.

```
yarn install
```

3. Tener nest CLI installado

```
npm -g install @nestjs/cli
```

4.  Levantar la base de datos.

```
docker-compose up -d
```

## Stack usado

- MongoDB
- NestJS
